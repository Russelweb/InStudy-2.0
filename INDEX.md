## 🔍 Quick Reference

### Commands
```bash
# Verify setup
python verify_setup.py

# Start application (Windows)
start.bat

# Start application (Linux/Mac)
./start.sh

# Backend only
cd backend && uvicorn main:app --reload

# Frontend only
cd frontend && streamlit run app.py
```

### URLs
- Frontend: http://localhost:8501
- Backend: http://localhost:8000
- API Docs: http://localhost:8000/docs

### Configuration Files
- `backend/.env` - Backend environment variables
- `frontend/.env` - Frontend environment variables
- `backend/config.py` - Backend settings
- `frontend/.streamlit/config.toml` - Frontend theme

### Key Files
- `backend/main.py` - API entry point
- `frontend/app.py` - UI entry point
- `backend/services/rag_service.py` - Core RAG logic
- `backend/services/document_processor.py` - Document handling
