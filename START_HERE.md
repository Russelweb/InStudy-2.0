## 🚀 Installation in 3 Commands

```bash
# 1. Install Ollama and pull Llama 3
ollama pull llama3

# 2. Install backend
cd backend && pip install -r requirements.txt

# 3. Install frontend
cd frontend && pip install -r requirements.txt
```

## ▶️ Run in 1 Command

**Windows:**
```bash
start.bat
```

**Linux/Mac:**
```bash
chmod +x start.sh && ./start.sh
```

**Access:** http://localhost:8501

## 🎯 First Steps After Running

1. **Create Course** → Click "Courses" → Create "Machine Learning"
2. **Upload Document** → Go to "AI Tutor" → Upload `backend/test_ml.txt`
3. **Ask Question** → Type "What is machine learning?" → Get answer
4. **Try Flashcards** → Go to "Flashcards" → Generate 10 cards
5. **Take Quiz** → Go to "Quiz" → Generate 5 questions
