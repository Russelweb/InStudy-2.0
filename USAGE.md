# InStudy 2.0 Usage Guide

## Getting Started

### 1. Create a Course
1. Navigate to "Courses" in sidebar
2. Click "Create New Course"
3. Enter course name (e.g., "Machine Learning")
4. Click "Create Course"

### 2. Upload Study Materials
1. Select your course
2. Go to "AI Tutor"
3. Click "Upload Study Material"
4. Choose PDF, TXT, or DOCX file
5. Click "Process Document"
6. Wait for processing to complete

### 3. Ask Questions
1. In "AI Tutor" page
2. Type your question in chat input
3. Toggle "Explain Like I'm 12" for simplified answers
4. Press Enter or click send
5. View AI response with sources

## Features Guide

### AI Tutor
**Best for:** Understanding concepts, getting explanations

**How to use:**
- Ask specific questions about your materials
- Use ELI12 mode for complex topics
- Check sources to verify information
- Build on previous questions in chat

**Example questions:**
- "What is gradient descent?"
- "Explain backpropagation step by step"
- "How do neural networks learn?"

### Flashcards
**Best for:** Memorization, quick review

**How to use:**
1. Select course
2. Choose number of cards (5-20)
3. Click "Generate Flashcards"
4. Use navigation buttons:
   - Previous/Next: Navigate cards
   - Flip: Show answer
   - Shuffle: Randomize order

**Tips:**
- Generate multiple sets for variety
- Use shuffle mode for better retention
- Review regularly

### Quiz
**Best for:** Testing knowledge, exam prep

**How to use:**
1. Select course
2. Configure:
   - Number of questions
   - Difficulty level
   - Question type
3. Click "Generate Quiz"
4. Answer questions
5. Check answers in expandable sections
6. Submit for score

**Tips:**
- Start with Easy, progress to Hard
- Use Mixed type for comprehensive testing
- Review explanations for wrong answers

### Summary
**Best for:** Quick review, exam prep

**Styles:**
- **Short:** 3-5 sentence overview
- **Bullet Points:** Organized key points
- **Detailed:** Comprehensive coverage
- **Exam Revision:** Focus on testable content

**How to use:**
1. Select course
2. Choose summary style
3. Optionally specify document
4. Click "Generate Summary"
5. Download as TXT if needed

### Study Planner
**Best for:** Long-term preparation, organization

**How to use:**
1. Enter course name
2. Set exam date
3. List topics (one per line)
4. Click "Generate Study Plan"
5. Follow weekly schedule

**Tips:**
- Be realistic with exam date
- List all major topics
- Follow daily tasks consistently
- Use revision tips before exam

## Best Practices

### Document Upload
- Upload organized, well-formatted documents
- Use descriptive filenames
- Upload lecture notes, textbooks, slides
- One topic per document works best

### Asking Questions
- Be specific in your questions
- Reference concepts from your materials
- Ask follow-up questions for clarity
- Use ELI12 for difficult concepts

### Study Strategy
1. Upload all materials first
2. Generate summary for overview
3. Use AI Tutor for deep understanding
4. Create flashcards for memorization
5. Take quizzes to test knowledge
6. Follow study planner schedule

### Exam Preparation
**2 weeks before:**
- Upload all materials
- Generate comprehensive summaries
- Create study plan

**1 week before:**
- Daily flashcard review
- Take practice quizzes
- Ask clarifying questions

**Day before:**
- Review exam-style summaries
- Final flashcard review
- Take hard difficulty quiz

## Troubleshooting

### "No documents found"
- Ensure you've uploaded documents to the course
- Check that processing completed successfully
- Verify you're in the correct course

### Slow responses
- Large documents take longer to process
- First query may be slower (loading vector store)
- Check internet connection

### Poor quiz quality
- Upload more comprehensive materials
- Try different difficulty levels
- Ensure documents cover the topic

### AI gives generic answers
- Upload more specific course materials
- Ask more detailed questions
- Reference specific concepts from your notes
